Overview

Financial Tracker is a Flask-based web application designed to help you manage your investment portfolio with ease. It fetches real-time stock quotes and news from multiple API providers, calculates optimal bi-weekly allocations, and provides comprehensive portfolio analytics. Whether you're a seasoned investor or just starting out, Financial Tracker gives you the tools to make informed investment decisions.

Key Features

Dashboard

Get a comprehensive overview of your portfolio with real-time market values, gains/losses, and performance metrics at a glance.

Portfolio Management

Track all your holdings with detailed information including shares, average cost, current prices, and profit/loss calculations.

Smart Allocation

Intelligent bi-weekly investment allocation based on your target percentages and current portfolio balance.

Stock Research

Access real-time stock quotes and latest news articles for all your holdings and potential investments.

Multi-API Support

Automatic fallback between Finnhub, Polygon.io, and Alpha Vantage APIs for reliable data access.

Excel Export

Export your complete portfolio analysis to Excel workbooks with multiple sheets and formatted data.

Getting Started

1. Clone the Repository

git clone https://github.com/YOUR-USERNAME/FinancialTracker.git
cd FinancialTracker

2. Install Dependencies

pip install -r requirements.txt

3. Configure API Keys

Create a .env file in the root directory: 

# Get free API keys from the providers
FINNHUB_API_KEY=your_finnhub_api_key
POLYGON_API_KEY=your_polygon_api_key
ALPHA_VANTAGE_API_KEY=your_alpha_vantage_api_key
SECRET_KEY=your_flask_secret_key

4. Run the Application

python app.py

Open your browser and navigate to http://localhost:5000

5. Command-Line Tool (Optional)

Generate Excel reports directly from the command line: 

python invest_plan.py --contribution 1000 --provider finnhub

Detailed Features

Configuration

Customize your portfolio allocation and holdings by editing config.json: 

{
  "allocation_buckets": {
    "Core / Tech": {
      "percentage": 25,
      "tickers": ["AAPL", "MSFT", "GOOGL"]
    },
    "Growth / AI": {
      "percentage": 30,
      "tickers": ["NVDA", "AMZN", "TSLA"]
    }
  }
}
